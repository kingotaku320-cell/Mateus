Bem-vindo ao meu site!

Este é um projeto simples criado para demonstrar a integração de HTML e CSS, seguindo um roteiro passo a passo. O objetivo é praticar a estruturação de uma página web básica com cabeçalho, conteúdo principal, destaques e rodapé.

Destaques do Projeto

Estrutura HTML

Usamos tags semânticas como `header`, `main`, `section`, `nav` e `footer` para organizar o conteúdo.

Estilização CSS

Aplicamos estilos básicos, como cores de fundo, fontes e layouts flexíveis para uma melhor visualização.

Responsividade Inicial

O layout utiliza uma abordagem básica para garantir que o conteúdo se ajuste a diferentes tamanhos de tela.
